# StackBlitz

## Importar

1. Use **Upload Project** e selecione o ZIP completo.
2. Aguarde `npm install` e `npm run dev`.
3. Crie um arquivo `.env.local` na raiz com:

```text
VITE_GAME_URL=https://ENDERECO-DO-APP.netlify.app
```

4. Reinicie o terminal do StackBlitz caso a variável não seja reconhecida imediatamente.

Sem essa variável, a landing abre normalmente e mostra uma orientação no espaço reservado para a demonstração.

O app não está mais copiado dentro deste projeto. Ele é carregado do endereço público informado, permitindo atualizações independentes.
